Module blink

Lua binding to the Blink1 library.

The main function in this library allows you to create Lua object bound to a blink(1) so you can control it with Lua code.

NOTE: This library only implements the Mk 2 versions of the various functions.

open ([devspec])
Opens a blink(1) device.

Create a userdata bound to a particular device. If this function is called without a parameter, it returns a userdata bound to the first device it finds.

Otherwise it attempts to bind to the specified device. A particular device is specified either by an integer ID or a string of 8 hexadecimal characters. The integer ID must be between 0 and n-1 (where n is the number of attached devices). The hex string is the serial number of a particular device.

Parameters:

  • devspec string or int device ID or serial number (optional)

Returns:

    userdata object bound to a device

Raises:

error if TBD

Device:fade (millis, red, green, blue, n)
Fades device to given RGB over given number of milliseconds.

Parameters:

  • millis int the fade duration
  • red int the red component [0-255]
  • green int the green component [0-255]
  • blue int the blue component [0-255]
  • n int which LED to adjust; 0 - both; 1 - top; 2 - bottom
